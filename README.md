# Weather Data Analysis using Python

## Project Overview

This project analyzes a weather dataset using Python. It includes data cleaning, exploratory data analysis (EDA), and data visualization to understand weather patterns such as temperature, humidity, precipitation, and wind speed.

## Objectives

- Load and clean the dataset
- Explore weather data
- Calculate basic statistics
- Create visualizations using Matplotlib
- Find meaningful insights from the data

## Dataset

The dataset contains **1,000,000 weather records** with the following columns:

- Location
- Date
- Temperature
- Humidity
- Precipitation
- WindSpeed

## Technologies Used

- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Analysis Performed

- Data Cleaning
- Descriptive Statistics

- Highest Humidity by Location

## Visualizations

- Bar Chart: Highest Temperature by Location
- Bar Chart: Highest Humidity by Location
- Histogram: Temperature Distribution in New York

## Key Insights

- The dataset contains 1 million weather records.
- Maximum temperature is around **40°C** across all locations.
- Maximum humidity reaches about **90%** in all locations.
- Temperature values range from **-10°C to 40°C**.
- The dataset is clean and suitable for analysis.

## Conclusion

This project demonstrates the complete data analysis process, including data cleaning, analysis, visualization, and reporting using Python.
